Who are you?

My name is Cass. Sometimes I make things. I make websites, 3D models, and games. But usually websites. I use a variety of frontend and backend tools, and enjoy learning new tools. I don't like bad software. Let's make something good instead.

What's This?

This is my portfolio website, designed and developed from scratch. No frameworks, libraries, or plugins, and the only tools I used to make this site are 11ty and Nunjucks, to make content that exists on multiple pages easier to update, as well as SASS for some helpful styling features, like minor colour adjustments for things like the glow you see on certain elements. The below ticker is done with just CSS, and anywhere you see my projects or work being displayed is custom JavaScript fetching and displaying content from an external file.

I did this because I wanted this site itself to be a showcase of my fundamental skills in web development, and not just a showcase of the ways that tools can make cool stuff easier and faster.
